## More Information

Buttons on focus MUST have a visual indicator of focus state. This doesnt mean it HAS to have an outline.
Anything that will pass the WCAG guideles will work. Buttons must have an acessibile name. Screen readers
MUST read out content when a button is focused. Buttons SHOULD have related text (ie learn more should go to somewhere to learn more).
Buttons SHOULD be able to stand on there own. This can be done in various aria ways.
